# 5. Zırhlı tugayı mekanik işleri proje takibi

React tabanli santiye ilerleme takip paneli.

## Calistirma

```bash
npm install
npm run dev
```

Uygulama varsayilan olarak `http://127.0.0.1:5173/` adresinde acilir. Port doluysa Vite baska bir port verebilir.

## Giris Bilgileri

- Super Admin: `admin` / `admin123`
- Formen: `formen1` / `formen123`

## Ozellikler

- PDF harita uzerinden tiklanabilir bina bolgeleri
- Admin icin bina, kullanici, izin ve renk araligi yonetimi
- Formen icin izinli binalarda is kalemi bazli ilerleme talebi
- Admin onay, revize ve fotografli talep takibi
